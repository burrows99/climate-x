Challenge

July 2025

Dear Candidate

Introduction

Time: 1-2 hours

Format: Written response (1–2-page report)

Submission: PDF document

Scenario:

An international asset manager approaches Climate X with a request to evaluate a selection of real estate investments that it is considering in the UK.  

The client has asked for a report summarising risk of the aggregate portfolio and any notable asset-level considerations. 

Instructions:

You are to run the API script for these 10 assets and produce a 1–2-page report based on the returned data outputs. 

The report should answer the following questions: 

What are the concentrations of risk in the portfolio if the investors purchase all 10 assets?

Which asset/s are highest risk? Why? 

Which asset/s would you recommend? Why?

How might risk change over time? 

What assumptions/limitations might there be in these outputs? 

Anything else that might be relevant to the client’s due diligence process.

Assessment Criteria

Ability to interpret API data accurately and extract relevant insights.

Ability to frame findings in the context of business outcomes.

Notes

Open-book assessment

Data and an API guide, as well as your API key will be provided separately.
